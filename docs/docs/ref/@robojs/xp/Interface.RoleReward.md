# Interface: RoleReward

Role reward configuration

## Example

```ts
{ level: 10, roleId: '1234567890123456789' }
```

## Properties

### level

```ts
level: number;
```

Level at which this role is awarded (must be positive integer)

***

### roleId

```ts
roleId: string;
```

Discord role ID (18-19 digit snowflake)
